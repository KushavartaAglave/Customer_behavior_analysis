Customer Shopping Behavior Analysis
Overview

Customer Shopping Behavior Analysis is an end-to-end Data Analytics project designed to analyze customer purchasing patterns, spending behavior, subscription trends, and product performance using real-world transactional data.

The project combines Python, SQL, MySQL, and Power BI to transform raw shopping data into meaningful business insights that support data-driven decision-making.

This project demonstrates practical skills in:

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
SQL Business Querying
Data Visualization
Dashboard Development
Business Intelligence & Reporting
Business Problem

Retail businesses generate large amounts of customer transaction data, but extracting actionable insights from this data can be challenging.

This project aims to answer important business questions such as:

Which customer groups generate the highest revenue?
How do discounts affect purchasing behavior?
Which products have the best customer ratings?
Are loyal customers more likely to subscribe?
Which age groups contribute the most revenue?
